Welcome to the Sustainability Report App

This is an OpenSource Artifact developed by Maria Davila from OFFIS Institute of Technology for the EDNA project. For more details, visit the project EDNA.

Objective: The objective of this application is to facilitate the report of carbon footprint for manufacturing companies.

The components of this project are built using the following resources:

  • Database: TimeScaleDB, developed by Postgresql. Learn more about Postgresql at their site.
  • Dashboard: Implemented using Grafana. Visit Grafana for more information.
  • Website: Developed using Angular Material.
