This is an OpenSource Artifact developed by Maria Davila from OFFIS Institute of Technology together with the project partners in EDNA. For more details, visit the project EDNA.

The components of this project are built using the following resources:

  • Database: TimeScaleDB, developed by Postgresql. Learn more about Postgresql at their site.
  • Dashboard: Implemented using Grafana. Visit Grafana for more information.
  • Website: Developed using the Angular Material tutorial from this GitHub repository.
